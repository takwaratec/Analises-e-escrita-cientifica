# Ficha Técnica: Plano de Validação Laboratorial para Bambu Estrutural

## 1. Identificação

**Título:** Plano de Validação Laboratorial da Takwara Tech para Bambu Estrutural  
**Campo de Aplicação:** Caracterização físico-mecânica, ensaios de durabilidade e desempenho de componentes estruturais de bambu tratado  
**Idealização:** Fábio Takwara  
**Escopo:** Roteiro completo de ensaios para validação técnico-científica da Tecnologia Takwara (tratamento térmico, impermeabilização, conexões mecânicas, compósitos)

## 2. Descrição

O plano estabelece protocolos para validação laboratorial dos elementos da Takwara Tech (TT): tratamento térmico com vapor d'água saturado + ácido pirolenhoso + sal marinho; impermeabilização com PU Vegetal; conexões mecânicas com aço flexível + PU Vegetal Expansivo; e compósitos de resíduos. Utiliza normas nacionais e internacionais como referência.

### Elementos Técnicos a Validar

- **Tratamento Térmico:** Exposição a ~90°C por 4-6h em câmara aquecida
- **Impermeabilização:** Camada protetora de PU Vegetal (base óleos vegetais)
- **Conexões Mecânicas:** Aço flexível + PU Vegetal Expansivo
- **Usinagem e Acabamento:** Pilares, vigas, tesouras, treliças
- **Compósitos de Resíduos:** Fibra de bambu + PU Vegetal prensados

## 3. Ensaios Prioritários

### Caracterização Físico-Mecânica do Bambu (antes e após tratamento)

| Ensaio | Parâmetros | Norma de Referência |
|--------|------------|---------------------|
| Densidade (seca e básica) | kg/m³ | ISO 22157 / NBR 16828-2 |
| Teor de Umidade | % | ISO 22157 |
| Absorção de Água e Inchamento | % | ISO 22157 |
| Compressão Paralela às Fibras | MPa | ISO 22157 / NBR 16828-2 |
| Flexão do Colmo (MOR/MOE) | MPa / GPa | ISO 22157 |
| Cisalhamento Paralelo às Fibras | MPa | ISO 22157 |
| Tração Paralela às Fibras | MPa | ISO 22157 |
| Dureza (resistência à penetração) | - | ISO 22157 |

### Ensaios de Durabilidade

- Resistência à decomposição (fungos e microrganismos) — ASTM D1413
- Resistência a insetos xilófagos (cupins e brocas)
- Resistência a intempéries (exposição simulada ou natural)

### Testes de Componentes e Conexões

- Capacidade de carga das conexões (tração, compressão, cisalhamento)
- Precisão dimensional de peças usinadas
- Resistência de compósitos (compressão, absorção de água, abrasão)

## 4. Aplicações

- Certificação técnica da Takwara Tech para uso estrutural
- Subsídio a projetos de engenharia e arquitetura com bambu
- Conformidade com normas ABNT NBR 16828 e ISO 22156
- Abertura de mercado para produtos de bambu tratado e compósitos
