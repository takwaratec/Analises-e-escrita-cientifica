## 3. RESUMO / SÍNTESE

Pesquisador pioneiro no uso de **poliuretano à base de óleo de mamona** para fabricação de painéis OSB e aglomerados com resíduos de madeira e fibras vegetais na USP São Carlos. Publicou dezenas de artigos exatamente sobre a mistura que o projeto Fábrica Modelo propõe: painéis estruturais + resíduos biológicos + PU de mamona. Uma carta de anuência dele no projeto FINEP garante chancela técnica indiscutível.
