## 3. RESUMO / SÍNTESE

Apresentar o padrão VERRA/VCS de certificação de créditos de carbono voluntários, suas metodologias aplicáveis a biomassa, biochar e construção, e o potencial para projetos com bambu.



Pesquisa documental nos padrões e metodologias publicados pela Verra (VCS, CCBS, SD VISta), incluindo VM0003, ARR e AMS-III.H.



| Campo | Dado |
|---|---|
| Sigla | VERRA / VCS |
| Nome | Verified Carbon Standard |
| Organismo | Verra |
| Ano | 2005 (VCS), 2007 (Verra) |
| Tipo | Padrão de créditos de carbono voluntários |
| Site | https://verra.org |



- VCS — créditos de carbono
- CCBS — co-benefícios socioambientais
- SD VISta — contribuição aos ODS
- Plastic Waste Reduction



| Metodologia | Aplicação |
|---|---|
| VM0003 | Biochar — pirólise de biomassa |
| ARR | Aflorestamento/Reflorestamento |
| AMS-I.C. | Energia renovável |
| AMS-III.H. | Recuperação de metano |

Processo: desenvolver/adaptar metodologia → validar (VVB) → registrar → verificar → emitir VCUs.



Gera créditos via substituição de clínquer (cinza/escória). Não há metodologia consolidada para substituição de concreto por bambu.



- **Biochar (VM0003)**: pirólise de bambu — sequestro de carbono
- **ARR**: plantio de Guadua — sumidouro
- **Energia de biomassa**: briquetes de bambu/açaí
- **Substituição de materiais**: exigir
