## Descrição

Esquema de certificação ENPlus para pellets de madeira, definindo classes de qualidade (ENPlus A1, A2, B), especificações técnicas (diâmetro, comprimento, teor de cinzas, poder calorífico) e requisitos para produção e comercialização.
