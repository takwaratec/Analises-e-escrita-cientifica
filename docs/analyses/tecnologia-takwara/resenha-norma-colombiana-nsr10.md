# Resenha: Norma Colombiana NSR-10 — Título G: Estructuras de Madera y Estructuras de Guadua

## 2. Resumo

A NSR-10 é o regulamento colombiano de construções sismo-resistentes. O Título G estabelece os requisitos de projeto estrutural para edificações de madeira e de guadua (bambu colombiano Guadua angustifolia Kunth). O Capítulo G.12 é especificamente dedicado às Estruturas de Guadua.

**Estrutura do Título G:**
- **G.1** — Requisitos Generales (alcance, definições, nomenclatura)
- **G.2** — Materiales (propriedades da madeira e guadua)
- **G.3 a G.11** — Dimensionamento de elementos estruturais, uniões, montagem
- **G.12** — Estructuras de Guadua (normas específicas para bambu)

**Principais definições e requisitos:**
- A norma complementa-se com as Normas Técnicas Colombianas NTC-5300 (Cosecha y postcosecha del culmo de guadua), NTC 5301 (Preservación y secado), NTC 5407 (Uniones) e NTC 5525 (Métodos de ensayo)
- A guadua é tratada como material estrutural legítimo, com requisitos de projeto, ensaios e controle de qualidade
- Inclui definições de termos técnicos: albura, amarre de continuidad, anisotropía, capacidad estructural, etc.
- Estabelece níveis de segurança comparáveis a edificações de outros materiais

## 3. Relevância para o Acervo

A NSR-10 é o marco regulatório mais avançado do mundo para construções de bambu, servindo como referência técnica e normativa para o desenvolvimento de regulamentações equivalentes no Brasil. O Título G demonstra que o bambu (guadua) pode ser tratado como material estrutural com requisitos de segurança sísmicos rigorosos. O capítulo G.12, especificamente sobre guadua, é um modelo para a futura norma brasileira de estruturas de bambu atualmente em elaboração na ABNT. A norma colombiana prova que o bambu é viável como material de construção em larga escala.
