# Ficha Técnica: O Bambu no Brasil — Visão Geral

## 1. Identificação

**Título:** Relatório Abrangente sobre o Bambu: Botânica, Geologia, Mercado, Utilizações, Engenharia, Sustentabilidade e Impacto Social  
**Campo de Aplicação:** Caracterização geral do bambu como recurso estratégico para construção, indústria e bioeconomia  
**Escopo:** Análise multifacetada do bambu: classificação botânica, evolução geológica, distribuição global, mercado, aplicações em engenharia

## 2. Descrição

O bambu (Bambusoideae, Poaceae) é uma gramínea lenhosa de rápido crescimento, conhecida como "aço verde". Com mais de 115 gêneros e aproximadamente 1.400 espécies, está presente em todos os continentes exceto Antártida e Europa. O Brasil possui a maior diversidade das Américas (~230 espécies nativas, 174 endêmicas).

### Classificação Botânica

| Nível | Nome |
|:------|:-----|
| Reino | Plantae |
| Divisão | Magnoliophyta |
| Classe | Liliopsida |
| Ordem | Poales |
| Família | Poaceae (Gramineae) |
| Subfamília | Bambusoideae |
| Tribo | Bambuseae (lenhosos) / Olyreae (herbáceos) |

### Origens Geológicas

- Origem da família Poaceae: Cretáceo Superior (~96 Ma)
- Diferenciação Bambusoideae: Eoceno Superior (~37 Ma)
- Diversificação principal: Mioceno (23,7–5,3 Ma)
- Distribuição moderna influenciada pelo soerguimento dos Andes e ciclos de Milankovitch

### Mercado Global (2024)

- Valor: USD 67–75 bilhões (projeção 2033: USD 102–117 bilhões)
- CAGR: ~4,75–5,0%
- Maior segmento: Madeira e mobiliário (>45%)
- Crescimento mais rápido: Alimentos (brotos) — CAGR 8,2%
- Têxteis: crescimento anual ~8%

## 3. Parâmetros Técnicos

| Propriedade | Faixa |
|-------------|-------|
| Resistência à Tração | Até 193 MPa (superior ao aço: 159 MPa) |
| Resistência à Compressão | 40–80 MPa (2–4x madeira comercial) |
| MOR (Flexão) | 76–168,6 MPa |
| MOE (Elasticidade) | 18–20 GPa |
| Densidade | 500–850 kg/m³ |
| Crescimento | Até 1 m/dia em algumas espécies |
| Maturidade Estrutural | 3–6 anos |
| Sequestro CO₂ | 12.000–17.000 kg/ha/ano |

### Engenharia do Bambu

- Bambu Laminado Colado (BLC) / LBL
- Bambu Scrimber (BS)
- Normas: ISO 22156 (projeto estrutural), ISO 22157 (caracterização), ISO 19624 (classificação)
- NBR 16828-1/2 (Brasil)

## 4. Aplicações

- Construção civil (estruturas, HIS, painéis, pisos)
- Mobiliário e design
- Têxteis (fibras antibacterianas)
- Papel e celulose (20% da matéria-prima global)
- Bioenergia (biomassa, carvão vegetal, pellets)
- Alimentação (brotos)
- Cosméticos (extratos)
