# Ficha Técnica: Caracterização Físico-Mecânica de Espécies de Bambu

## 1. Identificação

**Título:** Caracterização Físico-Mecânica de Espécies de Bambu para Aplicações Estruturais  
**Campo de Aplicação:** Engenharia estrutural, banco de dados para aplicativo de cálculo, projeto de estruturas de bambu  
**Gêneros Foco:** Guadua, Dendrocalamus, Phyllostachys, Bambusa  
**Escopo:** Compilação detalhada de propriedades físicas e mecânicas, fatores de influência (idade, posição, nós, origem geográfica) e normas aplicáveis

## 2. Descrição

Síntese abrangente de estudos de caracterização de bambu para aplicação em cálculo estrutural. Aborda densidade (gradiente radial e longitudinal), teor de umidade, retração anisotrópica e propriedades mecânicas (compressão, tração, flexão, cisalhamento) conforme normas ISO 22157, ISO 22156 e ISO 19624.

## 3. Parâmetros Técnicos

### Fatores de Influência

- **Idade:** Resistência ótima entre 3–5 anos (lignificação máxima)
- **Posição no Colmo:** Densidade e resistência aumentam base→topo e interno→externo
- **Nós:** Aumentam resistência à compressão; diminuem tração e flexão
- **Teor de Umidade:** Relação inversa com resistência; <15% ideal para uso estrutural
- **Origem Geográfica:** Afeta microestrutura e propriedades (ex: D. strictus seco > úmido)

### Normas de Referência

| Norma | Função |
|:------|:-------|
| ISO 22157:2019 | Métodos de teste para propriedades físicas e mecânicas |
| ISO 22156:2021 | Projeto estrutural (edifícios até 2 pavimentos, ≤7m) |
| ISO 19624:2018 | Classificação visual e mecânica de colmos |
| ASTM D4442 | Teor de umidade (secagem em estufa) |
| NBR 16828-1/2 | Estruturas de bambu (ABNT) |

## 4. Aplicações

- Base de dados para software de cálculo estrutural com bambu
- Dimensionamento de pilares, vigas, treliças e conexões
- Seleção de espécies para aplicações específicas
- Projetos de HIS, pavilhões, coberturas e pontes
