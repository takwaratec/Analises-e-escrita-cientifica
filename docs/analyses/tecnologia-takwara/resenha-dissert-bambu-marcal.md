## 2. Resumo

A dissertação realiza uma análise comparativa das principais normas técnicas internacionais que regulamentam o uso do bambu-colmo em estruturas prediais. São examinadas normas de países como Colômbia (NSR-10), Peru (E.100), Equador, Índia e China, comparando critérios de caracterização do material, propriedades mecânicas, métodos de ensaio, fatores de segurança, detalhamento de ligações e requisitos de durabilidade. O trabalho identifica convergências e divergências entre as normas e propõe diretrizes para subsidiar a elaboração de uma norma técnica brasileira para estruturas de bambu, contribuindo para a difusão segura e normalizada do material no país.

## 3. Relevância para o Acervo

Esta dissertação é de altíssima relevância para o Acervo Científico da Takwara Tech por consolidar o estado da arte da normalização técnica internacional para estruturas de bambu. O trabalho oferece base comparativa essencial para projetos de engenharia com bambu-colmo no Brasil, orientando escolhas de parâmetros de projeto, fatores de segurança e métodos de ensaio. É referência obrigatória para qualquer iniciativa de construção civil com bambu no contexto brasileiro.
